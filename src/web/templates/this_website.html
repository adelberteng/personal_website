About this website

Overview

Hi! thanks again, I am pretty glad you have interested in my website.
The reason why I built this website is because I want a place where can show my side projects and let people know more about me.

The website structure is shown above, this project is running on Google Compute Engine (e2-micro with 2 vCPUs, 1 GB memory), which is the compute instance on Google Cloud Platform. It builds with three different components (Proxy server, Web application and Database) and all of them are run inside the container.
The source code is available at Github.

Web Application

I mainly use Python for programming, so I decided to use Flask to be the web application framework and add Gunicorn to the front as a WSGI web server for improving performance.

In my website, in addition to have some static pages, also implement some functionalities to interact with backend system. You can sign up and login this website, browse the shopping page (that is dummy, of course) after login, pick up a product into your cart, view your cart item and finally set up an order. After you finish your testing, you can logout your account.
